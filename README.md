# 9 Turns

An asynchronous, turn-based 3√ó3 grid card game inspired by [Triple Triad](https://finalfantasy.fandom.com/wiki/Triple_Triad) and [spacetraders.io](https://spacetraders.io/). Backed by a fully API-driven server, players collect, trade, and battle cards, queue matches, and track progress via developer-friendly endpoints.

> This is a solo project provided **as-is**. For questions, contact [contact@9turns.com](mailto:contact@9turns.com).

---

## Overview

- **Cards**: Collect from nearly **1,000 unique cards**, each with its own stats and artwork  
- **Play**: Create players, manage decks, farm cards  
- **Battle**: Queue or join matches, place cards on a 3√ó3 board  
- **Campaigns**: Unlock and tackle campaign nodes  
- **API-First**: All functionality exposed via REST endpoints
